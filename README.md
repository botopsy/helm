# BotOpsy Helm Charts

[![Artifact Hub](https://img.shields.io/endpoint?url=https://artifacthub.io/badge/repository/botopsy-helm-charts)](https://artifacthub.io/packages/search?repo=botopsy-helm-charts)

BotOpsy Helm Charts is a collection of charts for [https://botopsy.github.io](https://botopsy.github.io) projects. The charts can be added using following command:

```bash
helm repo add botopsy https://botopsy.github.io/helm
```
